# 踩地雷 - 經典復古遊戲

![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)
![Platform](https://img.shields.io/badge/platform-Web-orange?style=flat-square)

一個使用原生 HTML5 Canvas 開發的復古踩地雷遊戲。

---

## 🎮 遊戲玩法

### 操作方式
| 按鍵 | 功能 |
|------|------|
| `左鍵` | 揭開格子 |
| `右鍵` | 標記地雷 |
| `R` | 重新開始 |

### 遊戲規則
- 揭開所有不含地雷的格子即獲勝
- 揭到地雷遊戲結束
- 數字代表周圍 8 格中含有的地雷數量
